# Manipulação de Listas em  Python

Este projeto em Python permite a manipulação de listas de inteiros com várias operações úteis como soma, média, procura de elementos, entre outras. A aplicação oferece um menu interativo que facilita a criação e a manipulação de listas de forma dinâmica.

## Funcionalidades

A aplicação permite realizar as seguintes operações:

1. **Criar Lista Aleatória**: Gera uma lista de números inteiros aleatórios de um tamanho definido pelo utilizador.
2. **Ler Lista**: Permite a inserção manual de uma lista de números.
3. **Somar Elementos**: Calcula e apresenta a soma dos elementos da lista.
4. **Calcular Média**: Calcula e apresenta a média dos elementos da lista.
5. **Maior Elemento**: Identifica e apresenta o maior número da lista.
6. **Menor Elemento**: Identifica e apresenta o menor número da lista.
7. **Verificar Ordem Crescente**: Verifica se os elementos estão ordenados em ordem crescente.
8. **Verificar Ordem Decrescente**: Verifica se os elementos estão ordenados em ordem decrescente.
9. **Procurar Elemento**: Permite ao utilizador procurar um elemento específico na lista e devolve a sua posição (ou -1 caso não exista).

## Como Usar

1. Ao iniciar o programa, será apresentado um menu com diferentes opções.
2. O utilizador pode escolher entre criar uma nova lista aleatória ou inserir uma lista manualmente.
3. Após a criação ou inserção da lista, pode escolher várias operações sobre a lista.
4. O programa termina quando o utilizador escolhe a opção 0.

## Estrutura do Código

- **menu()**: Função que apresenta o menu principal.
- **CRIAR_LISTA()**: Cria uma lista de números aleatórios.
- **ler_lista()**: Permite a inserção manual de uma lista.
- **soma()**: Calcula a soma dos elementos da lista.
- **media_lista()**: Calcula a média dos elementos da lista.
- **maior_lista()**: Determina o maior valor da lista.
- **menor_lista()**: Determina o menor valor da lista.
- **estaOrdenanda_por_ordem_crescente_lista()**: Verifica se a lista está em ordem crescente.
- **estaOrdenanda_por_ordem_decrescente_lista()**: Verifica se a lista está em ordem decrescente.
- **procura_um_elemento_lista()**: Procura um elemento específico na lista e devolve a sua posição.
