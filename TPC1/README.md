# RELATORIO TPC1
## explicação dos exercícios:
### Turtle (Nível 10)
**No Nível 10 do Turtle, o jogador usa laços e funções para desenhar uma forma complexa. O desafio é otimizar a movimentação da tartaruga, aplicando conceitos de repetição e precisão em coordenadas. Particularmente, o jogador construiu uma locomotiva a vapor bastante pormenorizada com duas carruagens.**

### Maze (Nível 10)
**No Nível 10 do Maze, o jogador usa estruturas condicionais e laços para guiar o personagem por um labirinto. O objetivo é resolver o puzzle, superando obstáculos de forma lógica e eficiente.**
