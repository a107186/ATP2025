# Gerenciador de Turma:

Este programa em Python permite gerir as informações de uma turma de forma simples e eficiente. Possibilita:

- **Criar** uma nova turma ou carregar de um ficheiro.
- **Adicionar alunos** com validação de dados.
- **Consultar e listar alunos**.
- **Salvar e carregar dados** em ficheiros para 

## Funcionalidades

- Navegação interativa através de um **menu**.
- Validação de IDs e notas para evitar inconsistências.
- Armazenamento de dados



Este programa executa de maneira prática a organização de informações de turmas.
